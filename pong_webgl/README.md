# Projeto Ping Pong 2D

Projeto de game 2D para matéria de Computação Gráfica.

O projeto consiste na recriação de um jogo pong em D2 em uma página web.

## Tecnologias

As tecnologias utilizadas no projeto foram:

- WebGL
- Javascript
- HTML
- CSS
